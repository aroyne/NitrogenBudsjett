# Subpool: Aquaculture (HY.AC)


### Flows that are zero or neglected:

* **HY.AC-MP.FP-Freshwater fish and seafood-Nmix**, **HY.AC-HY.SW-Waste feed-Nmix** and **HY.AC-HY.SW-Excretia-Nmix** are set to zero because practically all aquaculture in Norway is done in coastal waters.
* **HY.AC-AT.AT-Emissions-NH3** is set to zero assuming negligible ammonia emissions from these coastal marine cages.
