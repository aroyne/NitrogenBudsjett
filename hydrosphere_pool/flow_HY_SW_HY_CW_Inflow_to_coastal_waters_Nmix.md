# Inflow to coastal waters

![HY.SW-HY.CW-Inflow to coastal waters-Nmix](../output_files/plots/HY_SW_HY_CW_Inflow_to_coastal_waters_Nmix.png)

### Flow Description
Found from data supplied by NIVA, produced in the TEOTIL3 model Sample et al. (2024). Before 2013 we have used values from table 7.2 in Sample (2025). These values includes wastewater discharge, so to avoid double counting we subtract the flow *PR.WW-HY.CW-Treated wastewater discharge-Nmix* where we have already assigned all treated wastewater discharge to CW.

### References

* Sample, J. E. (2025). *Kildefordelte tilførsler av nitrogen og fosfor til norske kystområder i 2023 – tabeller, figurer og kart*.
* Sample, J. E., Jackson-Blake, L., Vogelsang, C., & Kaste, Ø. (2024). *TEOTIL3: En modell for beregning av kildebaserte tilførsler via elver og direktetilførsler til kyst*.
