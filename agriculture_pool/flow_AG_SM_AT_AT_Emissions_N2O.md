# N2 emissions from denitrification

![AG.SM-AT.AT-Emissions-N2](../output_files/plots/AG_SM_AT_AT_Emissions_N2O.png)

### Flow Description
Schäppi et al. (2025) recommends using a value of 14 kgN/ha/year for denitrification if no other data are available.
