# Urban Overland Flow

![HS.HS-HY.SW-Overland flow-Nmix](../output_files/plots/HS_HS_HY_SW_Overland_flow_Nmix.png)

### Flow Description
**HS.HS-HY.SW-Overland flow-Nmix** is a flow that has been added to account for runoff from urban areas. Some of this may actually end up directly in CW, but we have not been able to separate the two. Data are supplied by NIVA, produced in the TEOTIL3 model (Sample, 2024). For the period 1990-2013, we have used TEOTIL data (sample_teotil_data_2025) for nitrogen from urban areas that reach the coast, and used a retention rate of 5% which is consistent (with a 7% standard deviation) with the TEOTIL3 data from NIVA. The constant values for 1990-2012 is what is reported by Miljødirektoratet.
