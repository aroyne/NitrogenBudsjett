# Aquaculture Feed Import

![RW.RW-HY.AC-Aquaculture feed import-Nmix](../output_files/plots/RW_RW_HY_AC_Aquaculture_feed_import_Nmix.png)

### Flow Description
We assume a constant import fraction of 0.92 as given by Aas (2022) for the year 2020. The amount of feed used is based on the amount of fish produced, calculated using data from Fiskeridirektoratet (2025)on sold farmed fish, assuming average protein (N) retention of 35,75 % (Aas, 2022), 2.8 % nitrogen content in fish and shellfish  Schäppi (2025), p. 254) and 3% feed waste Wang (2013).
