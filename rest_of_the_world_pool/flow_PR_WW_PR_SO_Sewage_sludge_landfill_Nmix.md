# Sewage Sludge to Landfill

![PR.WW-PR.SO-Sewage sludge landfill-Nmix](../output_files/plots/PR_WW_PR_SO_Sewage_sludge_landfill_Nmix.png)

### Flow Description
Taken from SSB table 05279 *Avløpsslam, etter slamdisponering, statistikkvariabel, år og region*, including both sludge that is landfilled and sludge used for top cover on landfills Schäppi et al. (2025). For years 1993-2001 we use data from the SSB Naturressurser og miljø series. For years 1990-1992 we use the average value of the 1993-1995. We use a N content of 2.6 % as given in Table 54 in Schäppi et al. (2025).

### References
