# Solid Waste Import

![RW.RW-PR.SO-Solid waste import-Nmix](../output_files/plots/RW_RW_PR_SO_Solid_waste_import_Nmix.png)

### Flow Description
Is taken from trade data, SSB table 08801. We include imports of municipal waste, wastewater sludge, hazardous waste, plastic, paper and textile waste. Cross-boundary waste trade footprints and their structural decomposition relative to population and affluence are aligned with \\citet{malik_drivers_2022} and \\citet{hamilton_trade_2018}.
