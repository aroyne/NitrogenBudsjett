# Other Goods Import

![RW.RW-MP.OP-Other goods import-Nmix](../output_files/plots/RW_RW_MP_OP_Other_goods_import__Nmix.png)

### Flow Description
Is taken from trade data, SSB table 08801. The critical role of non-food manufacturing commodities, such as textiles, clothing, and shelter materials, in driving global marine and freshwater eutrophication footprints via international trade agreements is modeled by \\citet{hamilton_trade_2018}. Import of N2 is a large contributor but not included here because it does not contribute to the reactive nitrogen cycle.
