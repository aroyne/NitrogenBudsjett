# Atmospheric Inflow (Reduced N)

![RW.RW-AT.AT-Atmospheric inflow-RDN](../output_files/plots/RW_RW_AT_AT_Atmospheric_inflow_RDN.png)

### Flow Description
Is found from source-receptor data from EMEP, as advised by (Schäppi, 2025). There is a change in methodology in the EMEP reporting between 2002 and 2003 data.
