# Mineral Fertilizer Import

![RW.RW-AG.SM-Mineral fertilizer import-Nmix](../output_files/plots/RW_RW_AG_SM_Mineral_fertilizer_import_Nmix.png)

### Flow Description
Is taken from FAOSTAT Fertilizer by nutrient FAO (2025). Because anhydrous ammonia is not used directly as fertilizer in Norway, it is not counted as a fertilizer in this particular FAO statistic. We therefore include NH3 import in the flow **RW.RW-MP.OP-Other goods import-Nmix**.
