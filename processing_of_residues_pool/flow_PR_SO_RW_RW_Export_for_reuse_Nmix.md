# Export for Reuse

![PR.SO-RW.RW-Export for reuse-Nmix](../output_files/plots/PR_SO_RW_RW_Export_for_reuse_Nmix.png)

### Flow Description
**PR.SO-RW.RW-Export for reuse-Nmix** is exported used textiles. Data taken from trade data, SSB table 08801, and follows global non-food trade patterns described in (Hamilton, 2018).

### References

* Hamilton, H. A., Ivanova, D., Stadler, K., Merciai, S., Schmidt, J., van Zelm, R., Moran, D., & Wood, R. (2018). *Trade and the role of non-food commodities for global eutrophication*. Nature Sustainability. [https://doi.org/10/gnq4j3](https://doi.org/10/gnq4j3)
