# Material Recycling

![PR.SO-MP.OP-Recycling-Nmix](../output_files/plots/PR_SO_MP_OP_Recycling_Nmix.png)

### Flow Description
**PR.SO-MP.OP-Recycling-Nmix** is found from SSB tables 05281 “Avfallsregnskap for Norge (1 000 tonn), etter statistikkvariabel, behandlingsmåte, materialtype og år “ (1995-2011) and 10513 “Avfallsregnskap for Norge (1 000 tonn), etter materialtype, statistikkvariabel, år og behandlingsmåte” (2012-2023)), using N content values from in \/citet{schappi_annexes_2025}. We have not included the categories sludge, garden waste and wet organic material reported as being assigned to material recycling, because this use is rather for soild production or fertilizer and does not belong in the MP.OP subpool.
