# N2 Emissions (Wastewater)

![PR.WW-AT.AT-Emissions-N2](../output_files/plots/PR_WW_AT_AT_Emissions_N2.png)

### Flow Description
**PR.WW-AT.AT-Emissions-N2** is found by using data on N emissions and removal rates from wastewater treatment plants equipped with nitrogen removal. The dynamics of losing significant shares of excreted nitrogen as inert N2 via WWTP denitrification are detailed in \\citep{starck_fate_2023} and \\citep{fowler_global_2013}. Where specific data were missing we assumed a default 70 % removal rate.
